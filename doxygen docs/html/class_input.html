bool Input::GetKeyPress ( unsigned int  keyCode)
static

Returns whether a key is pressed.

Returns true if the specified key is held down, and false if it is not. Specify the key with GLFW_KEY_<character>. E.g. GLFW_KEY_Q indicates the button for 'Q' on the keyboard.
