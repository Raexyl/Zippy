Detailed Description

Application class.

This class is intended to be derived from (see Game.cpp for an example.) The game code should be written inside overridden methods of this class. The class that derives from App and holds the game's code, should be instantiated in GetApp().

◆ OnEnd()

void App::OnEnd ( )
virtual

Automatically called after App::Quit().

Override this method in your App derivative, and it's content will be executed once after App::Quit() runs. This is where you should prepare your application to exit (e.g. clean up memory, de-allocate resources.)

Automatically called to render a frame.

Override this method in your App derivative, and it's content will be executed once per frame, after App::OnUpdate(). This is where you should run all of your rendering-related code.

void App::OnUpdate ( )
virtual

Automatically called once for each frame.

Override this method in your App derivative, and it's content will be executed during each frame, before App::OnRender(). This is where you should run all of your non-rendering-related game code.
